AI Interview Question Generator - InterQ
AI Interview Generator is a smart, personalized interview preparation platform that dynamically generates role-specific interview questions and suggested answers using Groq AI. It helps job seekers practice effectively with customized question sets based on job roles, experience, company type and skill level.

Tech Stack
Frontend:	React, Tailwind CSS
Backend:	Node.js, Express.js
AI Engine:	Groq AI (Fast and reliable)
Database:	MongoDB

Key Features
Job-role based question sets
AI-generated sample answers using Groq
Question difficulty selection (Easy/Medium/Hard)
Dynamic follow-up and clarification questions
Secure and scalable backend using Express and MongoDB
